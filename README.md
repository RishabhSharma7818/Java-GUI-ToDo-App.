# Java Swing To-Do App

A simple desktop To-Do list application built with Java Swing.

## Overview
This application demonstrates basic desktop GUI development using Swing. You can add tasks, delete a selected task, clear all tasks, and save/load tasks to/from a simple text file (`tasks.txt`).

## Features
- Add task (press Enter or click **Add**)
- Delete the selected task
- Clear all tasks (confirmation dialog)
- Save tasks to `tasks.txt`
- Load tasks from `tasks.txt`
- Simple and readable code (single-file Java Swing app)

## Files
- `ToDoApp.java` — main Java source file
- `tasks.txt` — optional file created after saving (auto-created by Save)

## How to run (command line)
1. Make sure JDK is installed (Java 8+ recommended).
2. Compile:
   javac ToDoApp.java
3. Run:   
   java ToDoApp
